Язык запросов (Corpus Query Language)

Простые запросы

[ word = ”man” ] - точная словоформа

[ pos = ”NOUN” ] - часть речи

[ error = ”Grammatik” ] - тип ошибки

Регулярные выражения

[ word = ”man.*” ]

[ word = ”.*man” ]

[ word = ”.*man.*” ]

.* (точка и звездочка) означает, что на этом месте может быть любой набор символов (включая ноль символов)

Совокупность запросов

[ word = ”man” & error = ”Grammatik” ] - и

[ word = ”man” | error = ”Grammatik” ] - или

Условные операторы

[ word = ”man” ] - равно

[ word != ”man” ] - не равно
